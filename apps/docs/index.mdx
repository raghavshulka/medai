# MedAI

MedAI is a Turborepo monorepo for building a medical AI platform.

## Stack

- **Monorepo** — Turborepo + pnpm workspaces
- **Backend** — Express 5 on the Bun runtime, with BullMQ workers
- **Frontend** — Next.js 16 (App Router, React 19) + TanStack Query
- **Data** — PostgreSQL via Prisma 7 (pg driver adapter) + Redis (ioredis)
- **AI** — Vercel AI SDK with Anthropic (Claude) and OpenAI providers
- **Auth** — Better Auth: email/password, TOTP 2FA, bearer + JWT plugins
- **Tooling** — Biome, Vitest, Zod, Pino, PDFKit
