## Prerequisites

- [pnpm](https://pnpm.io) ≥ 10 (package manager)
- [Bun](https://bun.sh) ≥ 1.3 (API / worker runtime)
- [Docker](https://www.docker.com) (Postgres + Redis)

## Setup

<Steps>
  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
    This also runs `prisma generate` (postinstall).
  </Step>
  <Step title="Configure environment">
    ```bash
    cp .env.example .env
    # set BETTER_AUTH_SECRET and AI provider keys
    ```
  </Step>
  <Step title="Start Postgres + Redis">
    ```bash
    pnpm run db:up
    ```
  </Step>
  <Step title="Create the schema">
    ```bash
    pnpm run db:push
    ```
  </Step>
  <Step title="Run everything">
    ```bash
    pnpm run dev
    ```
    The API runs on `:4000`, the web app on `:3000`, and docs on `:3001`.
  </Step>
</Steps>

## Useful scripts

| Command | Description |
| --- | --- |
| `pnpm run dev` | Run all apps in watch mode |
| `pnpm run build` | Build all apps and packages |
| `pnpm run test` | Run the test suites |
| `pnpm run typecheck` | Type-check every workspace |
| `pnpm run db:push` | Sync the Prisma schema to Postgres |
| `pnpm run check` | Format + lint with Biome |
